Hot Weather by HotThemes is a weather forecast widget for WordPress.

== Description ==

Hot Weather plugin by [HotThemes](http://hot-themes.com/ "Hot Themes") helps you to inform your visitors about weather conditions for the selected city. Just enter your location WOEID and select units (Fahrenheit or Celsius).

== Installation ==

1. Install plugin as usually from WordPress.org, or download, and upload `plugin_hot_weather.zip` file your WordPress.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Publish the Hot Weather widget in one of available widget positions.
4. Click on the widget will open all Hot Weather options.
5. Detailed explanation of each plugin option and live demo can be found on [Weather](http://hot-themes.com/wordpress/plugins/weather/ "Weather") page.
